📊 GPU/NPU 信息检测

粘贴 nvidia-smi 或 npu-smi 命令输出，自动解析 GPU/NPU 型号、显存、驱动版本等信息。

✅ 验收标准参考

检测项合格标准备注
GPU 型号与合同清单一致对比 PO 清单
GPU 数量与合同清单一致所有 GPU 应被识别
显存容量与规格书一致A100-80G 应显示 81920MiB
驱动版本≥ 推荐版本参考兼容性矩阵
CUDA 版本≥ 客户需求版本通常 ≥ 11.8
ECC 状态已启用生产环境必须开启
GPU 温度（空闲）< 45°C确保散热正常
功耗范围在正常范围空闲时 < 额定 TDP 30%

🔗 PCIe 拓扑校验

粘贴 nvidia-smi topo -m 或 lspci | grep -i nvidia 输出，分析 GPU 间互联拓扑和带宽。

📐 理想拓扑参考

拓扑类型理想配置适用场景
NVLink 全互联所有 GPU 间 NVLink 直连大模型训练/推理（8卡 A100/H100）
NVSwitch 架构通过 NVSwitch 全互联H100/H200 集群
PCIe 直连GPU 直连 CPU PCIe Root推理卡（T4/L4/A10）
NUMA 亲和GPU 与 NIC 在同一 NUMA多 NIC 分布式推理

🔥 烤机日志分析

粘贴 GPU 压力测试工具输出（gpu-burn、DCGM 等），自动识别错误和异常。
